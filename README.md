SwiftMPNowPlayingInfoCenterSample
====================


DESCRIPTION
--------------------

Simple test application for `MPNowPlayingInfoCenter` and iCloud music items.
You can also see how to use IB size layout.


PLATFORM
--------------------

iOS 8.0b5 and later.


USAGE
--------------------

* Just build and run :-).
